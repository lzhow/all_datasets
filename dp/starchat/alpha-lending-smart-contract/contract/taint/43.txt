
The variable excess utilisation rate is a constant that is set when the contract is deployed. It is used to calculate the interest rate of the pool.

As the name suggests, the excessUtilisationRate is an excess amount of utilised funds that the protocol allows to be borrowed. This is to ensure that there is always enough funds to pay back all the borrowers.
