  3. FreeCodeCamp: Like CodeCademy, Free Code Camp offers courses on Python through interactive tutorials and projects.

 4. Udemy: UDEMY offers several courses in Python, including courses from industry experts and a comprehensive course on Flask, a popular web framework for Python developers.

5. YouTube: There's a large number of tutorials available on YouTube that can help guide you through the process of learning Python from scratch. Just search for "Python tutorial" and you'll find a wealth of options.

These are just a few of many great resources for learning howto code Python!
