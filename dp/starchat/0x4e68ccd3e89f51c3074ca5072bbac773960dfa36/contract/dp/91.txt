Python is an interpreted, high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation via the off-side rule.

"Python" is pronounced like the English word "benevolent."
The Python logo is based on a snake and a bug because Python was conceived as a successor to the ABC language and its associated interpreter, ABC. Guido van Rossum and the Python development team had a long and interesting chat about the logo and what it means. They also considered the possibility of a python mascot, but eventually settled on snake/bug because "why not?"
Guido's previous language work included ABC, Modula-3, C++, and Smalltalk. He worked on Python in  the late 80s and early 90 s as an effort to combine the best of these languages.
