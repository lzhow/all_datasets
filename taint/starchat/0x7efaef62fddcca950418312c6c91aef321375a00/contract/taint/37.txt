   What is an "incremental bundler" for web applications?

An incremental bundling tool or bundlers is any tool that takes a set of files that make up a web application and generates a single file that contains all the code, along with any other resources that are required to run the application.
Incremental bundl[ing] is to bund[le] as incremental optimization isto optimization. Incremental bundlings are used in development to make the development process more efficient. They are also used by web servers to serve the most efficient version of a page to a user, as only the changed portions of code need to be sent to the browser. This can result in significant performance improvements for the end user.[citation needed]
