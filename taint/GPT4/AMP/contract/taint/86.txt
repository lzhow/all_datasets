   What is an easy J Perm PLL algorithm?
A J-PLL algorithm is any algorithm that solves a Rubik's Cube using the J, P, and L moves. JPLL algorithms are easy to implement and can be used by novices to learn the basics of solving the cube. Some of these algorithms include:
- J’s algorithm
This algorithm starts by finding the edge pieces that can see the white center. Then, using J moves, these edge piece can align with the center piece. Next, we use P’ moves to permute the pieces around the edges
