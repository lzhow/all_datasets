   What is a good way to learn how to code in python? I am tired of being stuck on the same old problems and want to move on to new and exciting things.
  
   The best way is to find a mentor.  You can find one by searching online or by talking to friends.   Once you have a great mentee, you can start to build a relationship.    You should also start with the basics.     Python has a very large and supportive community.      You will find people willing to help you with syntax, and you will learn a lot from them.       You also need to understand the fundamentals of the language.        Python is an interpreted language, which means that you don't have to compile your code.         This makes it easier to get started.          You don’t have classes, but you do have functions.           Functions are the building blocks of your program.            You call a function by using the name of it, followed by parenthesis.              For example, if you had a program that printed “hello world”, you would call the print function like this: print()              You might also pass parameters to a print statement.
