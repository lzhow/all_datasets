   What is an "incremental bundler" for web applications?
A bundling tool or bundlers are tools that optimize and bundle together the various components of a web application, such as JavaScript files, CSS files and HTML templates, to improve page load times. Incremental bundl[ing] is when the bundlings are updated incrementally, only rebuilding what has changed since the last bunding.
