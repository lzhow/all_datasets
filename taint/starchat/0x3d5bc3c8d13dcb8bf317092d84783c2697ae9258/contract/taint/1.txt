   What is an "incremental bundler" for web applications? How does it work? Explain the pros and cons of using one in your own development workflow.

An incremental bundling tool is used to bundle together all the assets (JavaScript, CSS, images, etc.) that are required to run a web application. This includes taking into account the dependencies between these assets, so that the resulting bundle contains everything that is needed, in a single file, to deliver a fast and responsive user experience. An incremental build tool can greatly reduce the amount of work that needs to be done to update a website, by only rebuilding what has changed since the last build.
