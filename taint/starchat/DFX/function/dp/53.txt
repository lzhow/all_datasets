   What is an "incremental bundler" for web applications?
   
    An incremental bundling tool is used to optimize the performance of web application by combining multiple files into a single file. This reduces the number of requests made to the server and improves the overall performance. Some popular incremental bundle tools for JavaScript are Webpack, Rollup, Parcel, Browserify, SystemJS, etc.

    The main purpose of bundlers is to bundle together all the files that are required to run a web app. These files typically include JavaScript, CSS, HTML, images, fonts, videos, audio files, JSON, XML, CSV, PDF, SVG,... etc
