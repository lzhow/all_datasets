   What is an "incremental bundler" for web applications?

An incremental bundling tool or bundlers is any tool that takes a graph of dependencies between modules and generates a new graph where cycles have been broken and the modules have had their contents merged into a single file. Incremental bundlings are useful for bundlling web application code for deployment, as it allows only the changed code to be re-bundled, rather than the entire codebase.
