Python is an interpreted, high- level and general-purpose programming language. It was created by Guido van Rossum and first released in 1991. Python's design philosophy emphasizes code readability with the use of significant indentation via the off-side rule.

"Python" refers to both the programming languange and the community around it. Many Python users and developers are hailed as "batteries included" and love the idea of Python as a "glue" language that unites many different aspects of programming. This is in stark contrast to other programming languages such as C++ and Java, where the focus is on libraries and frameworks that provide specific functionality.
